# Team Member Card Elementor Widget

A custom WordPress plugin that extends Elementor page builder with a professional "Team Member Card" widget for displaying team member information.

## Description

This plugin adds a custom Elementor widget that allows you to create beautiful team member cards with customizable fields including name, role, biography, photo, and LinkedIn profile link. The widget features a modern card design with hover effects and full styling controls.

## Requirements

- WordPress 5.0 or higher
- Elementor 2.0.0 or higher
- PHP 7.0 or higher

## Installation

1. Go to WordPress Admin Panel > Plugins > Add New
2. Click "Upload Plugin" button
3. Choose the `CustomPlugin` folder (zip it first) and click "Install Now"
4. Click "Activate Plugin"
5. Ensure Elementor is installed and activated

## Usage

1. Edit any page or post with Elementor
2. Drag the "Team Member Card" widget from the widget panel to your page
3. Configure the widget settings:
   - **Name**: Team member's full name
   - **Role**: Job title or position
   - **Bio**: Brief biography or description
   - **Photo**: Upload team member's photo
   - **LinkedIn URL**: Optional LinkedIn profile link
4. Customize colors and styling in the Style tab
5. Save and preview your page
